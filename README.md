
# ESP32-C3 Digital Picture Frame

A Wi-Fi enabled digital picture frame built on ESP32-C3 with a 2.8" ST7789 touch display.

## Features
- JPEG slideshow from /pics/*
- HTTP sync from Debian server
- Only *.jpg files
- Sync twice per day
- Auto-delete removed images
- config.ini in flash
- Touch-based settings menu
